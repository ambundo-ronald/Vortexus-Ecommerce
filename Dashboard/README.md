# Nuxt 3 E-commerce Dashboard

A nice e-commerce dashboard built with Nuxt 3, featuring advanced data visualization, server-side rendering, and real-time analytics.



## Features

- 📊 Sales Dashboard
- 📈 Analytics Dashboard
- 👥 Orders, Products, Users
- 🖼️ Media Library / File Manager
- 📢 Ad Campaign Manager (soon)


## Quick Start

### Installation

```bash
# Install dependencies
pnpm install

# Or with npm
npm install
```

### Development

```bash
# Start development server on http://localhost:3000
pnpm dev

# Or with npm
npm run dev
```

### Production

```bash
# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Key Technologies

- **Nuxt 3** - Full-stack Vue framework
- **Nuxt UI** - Vue component library
- **nuxt-charts** - Chart visualization
- **TypeScript** - Type safety
- **Zod** - Schema validation
- **VueUse** - Composition utilities
